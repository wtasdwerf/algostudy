## 풀이

```python
for tc in range(10):
    n = int(input())
    arr = list(map(int, input().split()))
    result = 0

    for i in range(2, n-2):
         tmp = 0
        for j in range(i-2, i+3):
            if arr[i] <= arr[j]:
                break
            elif arr[j] < arr[i]:
                tmp = arr[i] - arr[j]
        result += tmp
    print(result)

```
